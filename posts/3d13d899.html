一. 表格标签

1.1 表格标签主要作用

表格主要用于显示、展示数据，因为可以让数据显示非常规整，可读性好。 表格不是来布局的，而是来展示数据的。
常用于获取到对应的数据库数据后，通过js赋值到相应的表格内

1.2 表格基本语法

  • table用于定义表格标签
  • tr用于定义表格的行，需要嵌套在table内
  • th用于定义表格内的列，需嵌套在tr内
  • td用来当表头进行展示，嵌套在tr内

1.3 表头单元格

一般单元格位于表格的第一行或第一列，表格单元格里面的文本内容 加粗显示。 <th> 标签表示 HTML 表格的表头部分（table head 的缩写） 表头单元格(<th>)也是单元格，常用于表格第一行，突出重要性，表头单元格里面的文字会加粗居中显示。

1.4 表格属性

稍作了解即可，后期通过css优化

属性名 属性值 描述
align left，center，right 表格对齐方式
border 1或者”” 设置表格边框是否展示，默认无边框
cellpadding 像素值 单元格与内容之间空白距离
cellspacing 像素值 单元格与单元格之间的间距
width 像素值 宽，可根据表格等比例缩放
height 像素值 高

1.5 表格结构标签

场景：表格可能很长，将表格分割为表格头部和表格主题两大部分。

加强语义化：<thead> 表示头部区域，<tbody> 表示主体区域，更好区分表格结构。

注意：

  1. <thead></thead>：定义表格头部，且内部必须有<tr>标签。
  2. <tbody></tbody>：定义表格主体，存放数据。
  3. 以上标签都放在 <table></table> 里面。

1.6 合并单元格

合并单元格方式：

  • 跨行合并：rowspan=”需要合并单元格数”
  • 跨列合并：colspan=”合并单元格数”

合并三部曲：

  • 确定跨行还是跨列
  • 声明合并标签，跨行取最上面单元格，跨列取最左面单元格
  • 删除多余单元格

二.列表标签

表格展示数据，列表则用来布局。 列表特点是整齐、整洁、有序，作为布局更方便自由。 根据使用场景不同，列表分为三大类：

  • 无序列表
  • 有序列表
  • 自定义列表

2.1 无序列表

<ul> 标签标识 HTML 页面项目中的无序列表，一般以项目符号呈现列表项。

语法格式：

注意：

  • 各个列表项内为并列关系

    • 中只能包含li，不能包含其他标签

    • 无序列表带有自己属性，但一般用css修改样式属性

    2.2 有序列表

    使用 <ol> 标签来定义有序列表，<li> 来定义列表项

    2.3 自定义列表

    <dl> 定义描述列表，与 <dt> 和 <dd> 标签配合使用。

    • dd都是为了用来解释dt的，但两者属于同级标签

    三.表单标签

    3.1 表头的作用

    收集用户信息，完成和用户的交互

    3.2 表头组成

    • 表单域
    • 表单控件，指其中的一个个元素
    • 提示信息

    3.3 表单域

    表单域是一个包含表单元素的区域。 使用 <form> 标签用于定义表单域，手机和传递用户信息。 form 标签会将它范围内的表单元素信息提交给服务器。

    目前而言，暂时不需要用表单域提交数据，只需要写上 form 标签即可。

    常用属性：

    属性 属性值 作用
    action url 地址 指定接受并处理表单数据的服务器的 url 地址
    method get/post 设置表单数据的提交方式，get常拼接在url后，用于获取信息，post用于提交信息
    name 名称 用于指定表单名称，区分同一个页面中的多个表单域

    注意：

    1. 写表单元素之前需要有表单域将其包含。

    3.4 表单控件

    在表单域中定义各种表单元素，允许用户在表单中输入或选择的内容控件

    3.4.1 input表单元素

    用于手机用户信息，可以输入文本字段，夫悬空，单选框，按钮等

    • type用来表示不同的控件类型

    type常用属性：

    属性值 描述
    button 定义点击按钮
    checkbox 定义复选框
    file 定义文件上传按钮
    hidden 定义隐藏的输入字段
    image 定义图像形式的提交按钮
    password 定义密码按钮，密码以密文展示
    radio 表单单选按钮
    reset 将已经选中的值进行重置
    text 定义输入字段，可指定长度

    input其他属性：

    属性 属性值 描述
    name 自定义 用于定义提交的时候key
    value 自定义 用于定义input元素的值
    checked checked input元素首次加载的时候被选中
    maxlength 正整数 用于定义字符最大长度

    注意：

    • 对于radio单选框，name指定为相同的，这样才能多选一，复选框也是同理
    • 单选框和复选框都可以是指checked属性，设置之后默认选中
    • 设置type为submit以后，表单数据会进行提交，供后端处理
    • button按钮，常用与和js一起使用

    3.4.2 label标签

    label主要用于给某个表单元素绑定对应的关联，比如点击文字或者超链接就可以关联到对应的多选按钮，自动转换焦点到表单元素上，提升用户体验

    • for必须与表单元素的id值相同

    3.4.3 select下拉表单元素

    select 标签设置多个选项让用户选择，节约页面空间

    • 至少包含一个option
    • 定义selected=selected，对应的下拉列表会默认选中该值

    3.4.5 textarea文本框表单元素

    • cols为每行字符数，rows为显示的行数
