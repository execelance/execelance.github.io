一.CSS简介

1.1 html简介

HTML 是网页的骨架，只关注内容的语义。例如<h1>表示大标题，<p>表示段落。

早期的时候，HTML 只能做一些简单的样式，网页非常丑，而且使 HTML 代码臃肿。

1.2 css-网页的装修师

CSS 是 层叠样式表（Cascading Style Sheets）的简称。

CSS 也是一种标记语言。

CSS 主要用于设置 HTML 页面中的文本内容（字体、大小、对齐方式）、图片的外形（宽高、边框样式、边距等）以及版面的布局和外观显示样式。·

HTML 呈现结构，CSS 决定样式，结构与样式分离。

格式：选择器及一条或多条声明

  • 选择器用于指定CSS样式，花括号内是该对象的具体格式
  • 属性和属性值以键值对形式存在
  • 多个键值对之间用；分开

1.3 css书写格式

  • 展开式进行书写
  • 属性名，选择器，属性关键字全部用小写
  • 空格规范，选择器后面加一个空格，属性后面加一个空格

二.CSS基础选择器

2.1 选择器的作用

选择器实际就是找到相应的标签

2.2 选择器分类

选择器分为基础选择器和复合选择器

  • 基础选择器：标签选择器，类选择器，id选择器，通配符选择器

2.3 标签选择器

直接用 HTML 标签名作为选择器，按标签名称分类，为页面某一类标签指定统一的 CSS 样式。

语法：

优点：标签选择器可以把某一标签全部选择出来，快速为同类型标签设置统一样式。

缺点 ：能设置差异化样式，只能选择全部当前标签。

2.4 类选择器

差异化选择不同标签，单独选一个或者某个标签。

语法：使用class属性来调用class类，样式用点定义，结构用类调用，一个或者多个

注意：

  • 类选择器用.类名定义
  • 小写，多个单词之间用-来连接
  • 不要用纯数字，中文

2.5 id选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。

HTML 元素以 id 属性来设置 id 选择器，用 # 来定义。

样式#定义，结构用id调用

注意：

  • id选择器可以为特定id的html指定特定样式
  • id属性只能在每个html文档中出现一次，后续调用无效

2.6 通配符选择器

语法：

  • 用*定义通配符选择器，选取页面中所有标签
  • 无需调用自动给所有标签
  • 用于清除所有元素内外边距

2.7 基础选择器总结

基础选择器 作用 特点 使用情况 用法
标签选择器 选中所有相同标签 不能差异化选择 较多 p {color: red}
类选择器 选中一个或多个标签 可以根据需求定制化选择 较多 .nav {color: red}
id选择器 一次只能选出一个标签 一个id属性只能在一个页面中进行使用 一般配合js进行使用 #id {color: red}
通配符选择器 选择所有标签元素 选择的太多，有的不需要 特殊情况使用 * {color: red}

三.css字体属性

3.1 字体系列

  • 多个字体之间用，隔开
  • 有空格隔开的多个单词构成的字体，加引号
  • 字体顺序是默认从前往后选有的，都没有用默认字体

3.2 字体大小

  • px是我们网页常用单位
  • 谷歌浏览器默认大小为16px
  • 可以给body指定整个页面字体大小

3.3 字体粗细

  • 实际开发常用数字来表示，400表示normal，700表示bold

3.4 文字样式

属性值 描述
normal 正常显示
italic 斜体
  • 很少给字体加斜体，通常是用来将斜体改为正常的

3.5 字体复合属性写法

  • 必须按照如上顺序书写，并且以空格隔开
  • 必须有font-size，font-family属性，其他可省略不写用默认值

3.6 字体属性总结

字体属性 表示 注意
font-size 字号 单位是 px
font-family 字体 按照团队约定来写
font-weight 字体粗细 400=normal，700=bold
font-style 字体样式 italic，normal，常用 normal
font 属性连写 顺序不能变，字体和字号属性必须有

四.css文本属性

CSS Text（文本）属性定义文本外观，比如颜色、对齐、装饰、缩进、行间距等。

4.1 文本颜色

  • 可直接定义颜色，或者使用#FFFFFF这样，工作中一般由美工提供取色使用，还有一种是使用RGB(255,0,0)
  • color用于定义文本颜色

4.2 对齐文本

text-align 属性用于设置元素内文本内容的水平对齐方式。

属性值 解释
left 左对齐（默认）
right 右对齐
center 居中对齐

4.3 装饰文本

text-decoration 属性规定添加到文本的修饰。可以给文本添加下划线、删除线、上划线等。

属性值 描述
none 默认，无装饰（最常用）
underline 下划线，a 标签自带（常用）
overline 上划线。（几乎不用）
line-through 删除线。（不常用）

4.4 文本缩进

text-indent定义段落首行缩进。

  • 可以使用px或者em，em会根据你设置的字体缩进两个字体的长度

4.5 行间距

line-height 设置行间的距离，行间距=文本高度+上间距+下间距

4.6 文本属性总结

属性 表示 注意
color 文本颜色 通常十六进制缩写
text-align 文本对齐 用于设置文字对齐方式
text-indent 文本缩进 记住 text-indent: 2em;
text-decoration 文本装饰 记住下划线和去除下划线
line-height 行高

五.css引入方式

按照 CSS 书写的位置不同，CSS 样式表可以分为三大类：

  1. 行内样式表（行内式）
  2. 内部样式表（嵌入式）
  3. 外部样式表（链接式）

5.1 内部样式表

将 CSS 代码写在 HTML 页面内部，单独放在一个 <style> 标签中。

  • <style> 理论上可以放在 HTML 文档中任何一个地方，一般放在 <head> 标签中。
  • 方便控制整个页面中的元素样式。
  • 并没有实现结构与样式完全分离

练习时使用，实际开发不用。

5.2 行内样式表

行内样式表（内联样式表）是在元素标签内部的 style 属性中设定 CSS 样式，适合修改简单样式

5.3 外部样式表

实际开发中都是外部样式表，适用于样式比较多的情况。 核心：样式单独写到 CSS 文件中，再将 CSS 文件引入到 HTML 页面中。
